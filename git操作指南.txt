在任何大操作之前都要git pull //拉取所有远程更新（提交，合并等）
保证自己工作时是在自己的分支上
在提交时
0.git pull
1.git add .// 将本地文件提交到本地仓库
2.git commit -m"提交说明"
3.git push //将本地仓库文件推送至远程仓库
合并：在提交后把自己的分支内容提交到主分支或者把主分支更新内容更新到自己的分支
0.git pull
0-1.git pull origin main //更新主分支
1.git fetch . 自己的分支名:main //将自己分支更新提交到主分支
2.git push origin main //将主分支的更新推到github上的远程仓库

从主分支拉取更新时
0.git pull //更新当前分支
0-1.git pull origin main //更新主分支
1.git merge main //从主分支拉取更新
2.git push //将当前分支的合并操作提交到远程仓库
切换分支
git checkout 分支名

初始化克隆仓库
git clone 仓库网址
